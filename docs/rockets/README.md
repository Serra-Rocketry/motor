# Foguetes

Documentação CAD dos foguetes da Serra Rocketry.

## Foguetes

| Foguete | Altitude | Motor | Status | Arquivos |
|---|---|---|---|---|
| [500m](./500m/) | 500m | SR500M | Em desenvolvimento | CAD, STEP, montagem, OpenMotor |
| [1000m](./1000m/) | 1000m | SR1000M | Em desenvolvimento | CAD, STEP, montagem, OpenMotor |
| [Commercial](./commercial/) | — | — | — | CAD, STL, gcode |

## Estrutura por foguete

| Subpasta | Conteúdo |
|---|---|
| `cad/` | Peças (`.SLDPRT`) |
| `assembly/` | Montagens (`.SLDASM`) |
| `step/` | Modelos STEP para interoperabilidade |
| `parasolid/` | Modelos Parasolid (`.x_t`) |
| `simulation/` | Análises de simulação |
| `openmotor/` | Simulações do motor (`.ric`) |
| `mold/` | Moldes (`.step`) |
