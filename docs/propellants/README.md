# Propulsores (Propelentes)

Dados de testes de queima e caracterização de propelentes sólidos.

## Testes

### Teste de Queima — 09/06/2023

Teste de 14 formulações de propelente com variações de:
- Proporção KNO3/Açúcar (50/50, 65/35, 70/30)
- Substituição de Açúcar por Dextrose
- Adição de óxidos metálicos (CuFe2O4, CoFe2O4, NiFe2O4, ZnFe2O4)
- Adição de Clorato de Sódio

Documentação: [teste_queima_2023-06-09.md](./teste_queima_2023-06-09.md)

GIFs dos testes em [gifs/](./gifs/).

### Teste de Granulometria — Nitrato de Potássio

Avaliação de diferentes granulometrias de KNO3 para otimizar o desempenho do propelente.

Documentação: [granulometry_test_description.md](./granulometry_test_description.md)
